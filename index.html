xFader jQuery Plugin - Alexandre Mattos

Plugin de slide de imagens com controle e autorun. Controle vertical ou horizontal. Simples e objetivo.

Baixar Plugin xFader


Para utilizar é bem simples.

1 - Inclua no head do seu código o jquery via google, você pode usar o que está abaixo

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>

2 - Inclua abaixo do jquery que você colocou, o link para o script do plugin xFader.

<script src="js/jquery.xfader.min.js"></script>

3 - Insira o css do plugin e coloque a imagem dentro da respectiva pasta, se não mudar a estrutura das pastas tudo irá funcionar, caso mude, terá que mudar o endereço dos caminhos e no css mudar o endereço da imagem.

<link href="css/xfader.css" type="text/css" rel="stylesheet" />

4 - Insira o html em uma ul li como exemplo abaixo:

<ul id="fader1">
<li>
<img src="img/example1.png" alt="example1" />
</li>
<li>
<img src="img/example2.png" alt="example2" />
</li>
...
</ul>

5 - Lá no head novamente insira a chamada do plugin referenciando a sua lista

<script type="text/javascript"> jQuery(document).ready(function(){ jQuery('#fader1').xFader(); }); </script>

1 - xFader puro

jQuery('#fader1').xFader();

2 - xFader com auto:off

jQuery('#fader2').xFader({auto:false});

3 - xFader com auto:true e duration:2000

jQuery('#fader3').xFader({auto:true, duration:2000});

4 - xFader com controls e duration:5000

jQuery('#fader4').xFader({auto:true, showControls:true, duration:4000});

5 - xFader com controle na vertical

jQuery('#fader5').xFader({control:1, showControls:true,});
